## 1. 디자인 방향

### 레퍼런스
- **한국외대** — 현재 가장 근접한 느낌, 깔끔함 ✅ 이 방향 유지
- 연세대 총동문회 — 배경은 좋지만 가운데 박스가 거슬림 (피할 것)
- 서울대 — 뒷배경 흐릿하게 블러 처리한 디자인
- 하버드 — 언론사/카드뉴스 스타일

### 메인 페이지 스타일
- **카드뉴스 스타일** — 메인을 시작할 때 한눈에 직관적으로 볼 수 있도록
- 팝업 **금지** — 메인 팝업 없음
- 중요 공지는 하단 고정 배너 or 상단 띠 배너로 **딱 1개**만

### 웹 가독성
- 사용자 연령대 높음 (어르신) → 폰트 크기 크게, 터치 영역 넉넉히
- 폰트: Pretendard (저작권 무료) — 현재 적용 중 ✅

## 2. 모바일 최우선 ★★★

> "모바일을 통해서 유입이 가장 큼" — 가장 중요한 요구사항

- **생일 축하 문자** 매일 150명 발송 → URL 클릭 → 모바일 웹 진입
- 문자 유입 사용자가 첫 화면에서 바로 원하는 걸 찾을 수 있어야 함
- 모바일에서 카드뉴스처럼 한눈에 보이는 메인 구성
- 시안 2~3개 제작 후 결정

---

## 3. UX 필수 규칙

### 새 창 열기 금지 ★★ (매우 중요)
- **모든 페이지 이동은 현재 창 내에서** — `target="_blank"` 사용 금지 (내부 링크)
- 뒤로가기 버튼으로 이전 페이지 복귀 가능해야 함
- 현재 Footer, QuickMenu의 외부 기관 링크들 → 외부 사이트만 예외적으로 허용 검토 필요

### 관리자 편의성 ★★ (매우 중요)
- 메인 배너/소식을 **쉽고 빠르게 수정** 가능해야 함
  - 예: 골프대회 배너로 바꾸고 싶을 때 관리자가 직접 교체
- 소식 올릴 때 간단하게 — 사진 + 제목 + 내용으로 충분

## 4. 핵심 기능 요구사항

### 4-1. 인하상회 ★★★ (매우 중요)
- 메인 페이지에서 눈에 잘 보이도록 별도 섹션
- **동문만** 가입 가능한 폐쇄형 운영
- 회원가입 시 인하상회 가입 여부 연동 (옵션 제공)

### 4-2. 카드 납부 (회비 / 협찬금) ★★ (중요)
- 토스페이먼츠로 **카드 납부 중간 대행**
- 회비 + 협찬금 모두 카드 결제 가능하도록
- 현재: 동창회에서 페이 승인이 느림 → 카드 납부로 방향 전환

### 4-3. 이메일 발송
- 기존: 7~8만 명에게 홈페이지 내 발송 기능 있었음
- 현재 RESEND_API_KEY 환경변수 준비됨
- 건당 5원 다이렉트샌드 대비 더 좋은 방안 검토 (Resend: 월 3천건 무료)

### 4-4. 문자 발송
- 매일 150건 생일 축하 문자 (건당 8원, 장문 23원)
- NHN Cloud SMS 연동 예정 (NHN_APP_KEY 환경변수 준비됨)
- 카카오 알림톡 연동도 검토 (채널 연동)

### 4-5. 카카오 채널 연동
- 카카오 채널 연동 예정
- 푸터 또는 GNB에 아이콘 추가 필요

### 4-6. 회원 전용 콘텐츠
- **회비납부현황** — 로그인한 회원만 열람 가능 (비회원 차단)
- **자유게시판** — 일단 구현, 운영 방식은 추후 결정
  - 옵션 A: 글 자동 게시
  - 옵션 B: 관리자 승인 후 게시 (권장)

### 4-7. 장학회 홈페이지 분리 ★ (중요)
- 장학회는 재단법인 → 동문회와 **반드시 별도 도메인**으로 분리
- 현재 `/scholarship` 페이지들 → 외부 링크(inhaasf.com)로 연결하는 방식으로 변경
- 현재 코드의 `/scholarship/*` 페이지들 제거 또는 리다이렉트 처리 필요

### 4-8. 배너 광고
- 따라다니는 형식(fixed/sticky)으로 소수 배치
- 많지 않게 — 1~2개 수준

## 5. 현재 코드에서 수정 필요한 것들

| 항목 | 현재 상태 | 해야 할 것 |
|------|-----------|------------|
| `target="_blank"` | Footer, QuickMenu 등 다수 사용 | 내부 링크 전수 제거 |
| 모바일 레이아웃 | 데스크탑 중심 | 모바일 퍼스트 재설계 |
| `/scholarship/*` 페이지 | 직접 구현됨 | inhaasf.com 외부 링크로 변경 |
| 팝업 | 없음 ✅ | 유지 |

| 카카오 채널 | 없음 | 아이콘/링크 추가 |
| 관리자 배너 수정 | `/admin/banners` 존재 | UX 개선 필요 |
